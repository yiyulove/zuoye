# 子弹与敌机 的碰撞检测

## 思路

前提：  isTouch  判断2个物体是否碰撞

1. 创建定时器  不断检测 是否存在碰撞
		isTouch(物体1，物体2)

2. 如果 子弹 与 敌机 碰撞了
	 【获取所有的子弹对象】  遍历所有子弹，1个和所有敌机都进行碰撞检测
	 【获取所有的敌机对象】

		子弹 爆炸（销毁）
		敌机 血量-1  ==> 如果血量为 0， 爆炸（销毁）
		
3. 如果 敌机 与 玩家碰撞了
		玩家爆炸 ==> Game Over

#### 保存所有的子弹、敌机对象
1. 创建的时候，添加进去
2. 子弹或者敌机 到达目标位置之后，就删除

实现方式用： 用 全局对象 存储 所有 子弹 和 敌机
